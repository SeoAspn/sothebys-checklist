Connect Your Tools

All three connections are required before you can use the checklist. Your tokens are stored locally in your browser only — they are never sent to WDR servers.

● Asana — live tasks ● Airtable — submissions log ● Google Sheets — backup log
Get your token: app.asana.com → Profile → My Settings → Apps → Manage Developer Apps → New access token. Name it "WDR Checklist".
Asana token is required to load live tasks.
Get your token: airtable.com/create/tokens → Create token → Name it "WDR Checklist" → Scope: data.records:read and data.records:write on base Master sheet.
Airtable token is required to save and share submissions.
Set up once: Go to script.google.com → New project → paste the WDR webhook script → Deploy as web app → copy the URL here. Show me the script ›
Google Sheets webhook URL is required as a backup submissions log.

All three fields must be completed. Contact Anna if you need help getting your tokens.
